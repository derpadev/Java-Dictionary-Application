# Java Dictionary Application

## Overview
This is a Java-based dictionary application built with **JavaFX**. It allows users to search for words, add new entries, edit existing entries, and delete words from the dictionary. The application supports reading from and saving to files, ensuring that all changes persist across sessions.

## Features
- **Search**: Quickly look up definitions of words.
- **Add/Edit/Delete**: Manage dictionary entries easily.
- **File I/O**: Load dictionary data from files and save updates.
- **Error Handling**: Custom exceptions handle invalid operations.
- **Interactive GUI**: User-friendly interface built using JavaFX components.

## Technologies Used
- Java 17+
- JavaFX
- File I/O
- Exception Handling
